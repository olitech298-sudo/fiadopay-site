Politica de Privacidade

FiaPay

Ultima atualizacao: 26 de fevereiro de 2026

A sua privacidade e importante para nos. Esta Politica de Privacidade descreve como o aplicativo FiaPay ("nos", "nosso" ou "aplicativo") coleta, usa, armazena e protege as informacoes dos usuarios ("voce" ou "usuario"). Ao utilizar o FiaPay, voce concorda com as praticas descritas nesta politica.

1. Informacoes que Coletamos

1.1 Dados da conta (via Login com Google)

Utilizamos exclusivamente o Login com Google (Google Sign-In) para autenticacao. Ao fazer login, recebemos:

  • Nome completo
  • Endereco de e-mail
  • URL da foto de perfil (opcional)

Nao temos acesso a sua senha do Google.

1.2 Dados fornecidos por voce

Durante o uso do aplicativo, voce pode fornecer voluntariamente:

  • Dados do negocio: nome do estabelecimento, endereco comercial
  • Dados de clientes: nome, telefone, e-mail e endereco dos seus clientes
  • Registros financeiros: valores de fiados, pagamentos, datas de vencimento, metodos de pagamento e descricoes de transacoes
  • Preferencias: moeda, formato de data, tema do app, configuracoes de notificacao, mensagem de cobranca personalizada

1.3 Dados de assinatura

Se voce adquirir o plano FiaPay Premium, coletamos dados da transacao fornecidos pelo Google Play Billing:

  • Token de compra e ID do pedido
  • Status e data de expiracao da assinatura
  • Status de renovacao automatica

Nao armazenamos dados de cartao de credito ou debito. Todo o processamento de pagamento e realizado pelo Google Play.

1.4 Dados armazenados localmente no dispositivo

Algumas configuracoes sao salvas localmente no seu dispositivo via SharedPreferences:

  • Preferencias de notificacao (horario, dias de antecedencia)
  • Status de autenticacao biometrica
  • Cache de status premium (para acesso offline)
  • Preferencias de tema e configuracoes financeiras

1.5 Dados que NAO coletamos

O FiaPay nao coleta:

  • Localizacao (GPS ou rede)
  • Contatos do dispositivo
  • Fotos, camera ou microfone
  • SMS ou historico de chamadas
  • Dados de outros aplicativos
  • Identificadores de publicidade
  • Dados de analytics ou rastreamento comportamental

2. Como Usamos suas Informacoes

Utilizamos os dados coletados exclusivamente para:

  • Fornecer o servico: gerenciar seus registros de fiados, clientes e pagamentos
  • Autenticacao: identificar voce de forma segura via Google Sign-In
  • Notificacoes locais: enviar lembretes de vencimento e cobrancas no seu dispositivo
  • Assinatura Premium: validar e gerenciar o status da sua assinatura
  • Exportacao de dados: permitir que voce exporte seus dados em PDF ou CSV
  • Seguranca: oferecer autenticacao biometrica para protecao adicional

Nao utilizamos seus dados para publicidade, marketing direcionado ou criacao de perfis comportamentais.

3. Armazenamento e Seguranca dos Dados

3.1 Infraestrutura

Seus dados sao armazenados de forma segura nos servidores do Supabase, uma plataforma de banco de dados em nuvem com infraestrutura hospedada na Amazon Web Services (AWS).

3.2 Isolamento de dados (Row-Level Security)

Implementamos Row-Level Security (RLS) no banco de dados, garantindo que cada usuario so possa acessar, editar e excluir seus proprios dados. Nenhum usuario consegue visualizar dados de outro usuario.

3.3 Comunicacao segura

Todas as comunicacoes entre o aplicativo e nossos servidores sao realizadas via HTTPS/TLS, com criptografia em transito.

3.4 Autenticacao biometrica

O FiaPay oferece autenticacao biometrica (impressao digital / reconhecimento facial) como camada adicional de protecao. Os dados biometricos sao processados exclusivamente pelo sistema operacional do dispositivo e nunca sao enviados aos nossos servidores.

4. Compartilhamento de Dados com Terceiros

Compartilhamos dados apenas com os servicos necessarios para o funcionamento do aplicativo:

Servico Finalidade Dados Compartilhados
Google Sign-In Autenticacao Token OAuth (nome, e-mail, foto)
Supabase Banco de dados e autenticacao Todos os dados do aplicativo
Google Play Billing Processamento de assinatura Token de compra para validacao

Nao vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins de publicidade ou marketing.

5. Permissoes do Aplicativo

O FiaPay solicita apenas as permissoes estritamente necessarias:

Permissao Motivo
Notificacoes Enviar lembretes locais de vencimento de fiados
Alarme exato Agendar notificacoes no horario configurado pelo usuario
Inicio apos reinicializacao Reagendar notificacoes apos o dispositivo ser reiniciado
Biometria Permitir desbloqueio por impressao digital ou reconhecimento facial
Internet Sincronizar dados com o servidor

6. Retencao de Dados

Mantemos seus dados enquanto sua conta estiver ativa. Voce pode, a qualquer momento:

  • Exportar seus dados: em formato CSV ou PDF diretamente pelo aplicativo
  • Excluir dados individuais: remover clientes, fiados ou pagamentos especificos
  • Excluir todos os dados: apagar permanentemente todos os seus registros
  • Encerrar sua conta: ao excluir sua conta, todos os dados associados (clientes, fiados, pagamentos e assinatura) serao permanentemente removidos dos nossos servidores

7. Seus Direitos (LGPD)

Em conformidade com a Lei Geral de Protecao de Dados (LGPD - Lei 13.709/2018), voce tem direito a:

  • Acesso: saber quais dados pessoais possuimos sobre voce
  • Correcao: solicitar a correcao de dados incompletos ou inexatos
  • Exclusao: solicitar a eliminacao dos seus dados pessoais
  • Portabilidade: exportar seus dados em formato estruturado (CSV/PDF)
  • Revogacao do consentimento: revogar a autorizacao para uso dos dados a qualquer momento
  • Informacao: ser informado sobre com quem seus dados sao compartilhados

Para exercer qualquer um desses direitos, entre em contato conosco pelo e-mail indicado ao final desta politica.

8. Protecao de Dados de Menores

O FiaPay nao e destinado a menores de 18 anos. Nao coletamos intencionalmente dados de criancas ou adolescentes. Caso tomemos conhecimento de que coletamos dados de um menor, tomaremos medidas imediatas para excluir essas informacoes.

9. Notificacoes

O FiaPay utiliza apenas notificacoes locais (processadas no proprio dispositivo) para lembretes de vencimento. Nao utilizamos notificacoes push de servidores remotos. Voce pode desativar as notificacoes a qualquer momento nas configuracoes do aplicativo.

10. Alteracoes nesta Politica

Podemos atualizar esta Politica de Privacidade periodicamente. Quando fizermos alteracoes significativas, notificaremos voce por meio do aplicativo. A data da ultima atualizacao sera sempre indicada no topo deste documento.

Recomendamos que voce revise esta politica periodicamente para se manter informado sobre como protegemos suas informacoes.
